# Phase 1: Systems Foundations - Resources

This directory contains key resources mentioned in the Phase 1 roadmap.

## Downloaded Resources

- **[What Every Programmer Should Know About Memory](./what_every_programmer_should_know_about_memory.pdf)** by Ulrich Drepper.
  - *Description:* A fundamental paper on CPU caches, memory controllers, and how to write cache-efficient code. Essential for systems engineering.
- **[NVIDIA CUDA C++ Programming Guide](./cuda_c_programming_guide.pdf)**

  - *Description:* The official guide for programming with CUDA. Covers the execution model, memory hierarchy, and programming interface.

## External Resources (Commercial Books)

The following books are highly recommended but must be purchased or accessed through official channels:

- **C++20 in Detail** by Bartłomiej Filipek
  - [Official Purchase Page (Leanpub)](https://leanpub.com/cpp20indetail)
  - *Note:* Focuses on modern C++ features that aid in high-performance application development.
- **Professional CUDA C Programming** by John Cheng
  - [Official Purchase Page (Wiley)](https://www.wiley.com/en-us/Professional+CUDA+C+Programming-p-9781118739327)
  - *Note:* A comprehensive guide to advanced CUDA techniques and optimization.
